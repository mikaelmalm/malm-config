# malm-config

This repository contains my personal configuration files (dotfiles) for various applications and settings.

## Usage
symlink this repository to `~/.config/`

ln -s ~/code/malm-config/.zshrc ~/.zshrc
ln -s ~/code/malm-config/alacritty.toml ~/.config/alacritty/alacritty.toml
ln -s ~/code/malm-config/starship.toml ~/.config/starship.toml

## Contents

- `alacritty.toml`: Configuration for the Alacritty terminal emulator.
- `starship.toml`: Configuration for the Starship prompt.
- `.zshrc`: Configuration for the Zsh shell.
